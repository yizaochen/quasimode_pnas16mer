# quasimode_pnas16mer
### Create a virtual environment
`conda create --name quasimode python=3.8`
### Activate virtual environment
`conda activate quasimode`
### Deactivate
`conda deactivate`
### Install quasimode package
`pip install -e .`
### Upgrade package
`pip install -e . --upgrade`
